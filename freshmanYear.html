Freshman Year Projects

Here you can find all of the projects that I've been working on during my freshman year!

Flappy Bird

For my Software Design Studio Course (CS 126) final, I decided to re-create the popular Flappy Bird Game in C++. For this project, I used the OfxGui library and other significant tools in OpenFrameworks.


Google CodeU Chat App

Over the summer, I participated in a program for aspiring software engineers called Google CodeU. Over the course of 12 weeks, I was able to collaborate with a team of peers to design and implement a web application using Java, JavaScript, HTML, and CSS. When working on this project my team also used various Google Cloud Platform APIs, including App Engine and Datastore. Our chat app allows users to connect through conversations that are started at different locations on a google map. We wanted the map to be the focal point of our app so we turned it into a way for a user to join or start a conversation. Our chat app also has some other features that include:

  • Activity Feed: A feed that shows all of the application's activity such as when users start conversations, write messages, or join the application.
  • Profile Pages: Pages that display a user's about me, profile picture, and a mini feed that includes all of the messages a user has sent. When logged in, users have the option to edit their about me and picture right from their profile page.
  • Text Styling: The ability to style text by using certain BBCode tags ([b]wow![/b] -> wow! ) or parse text to emojis (:cat: -> üê±). Our website also has a text styling guide available for those who aren't familiar with BBCode tags and would like to learn how to use them on our site.
